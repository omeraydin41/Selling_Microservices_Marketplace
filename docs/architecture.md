# Architecture

## Overview
This project is designed using a microservices-based architecture.

## High-Level Components
- API Gateway
- Authentication Service
- Product Service
- Order Service
- Payment Service
- Message Broker (RabbitMQ)

## Communication
- Synchronous communication via REST APIs
- Asynchronous communication via RabbitMQ events

## Infrastructure
- .NET-based services
- Docker for containerization
